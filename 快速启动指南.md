# 歧义侦探 - 快速启动指南

## 🚀 快速启动步骤

### 步骤1：安装依赖（首次运行）
打开命令提示符或PowerShell，执行以下命令：

```bash
# 安装Python包
pip install streamlit spacy nltk

# 下载spaCy英文模型
python -m spacy download en_core_web_sm

# 下载nltk数据
python -c "import nltk; nltk.download('punkt'); nltk.download('punkt_tab'); nltk.download('averaged_perceptron_tagger')"
```

### 步骤2：启动应用
有两种启动方式：

#### 方式1：使用启动脚本（推荐）
双击运行 `启动.bat` 文件

#### 方式2：命令行启动
```bash
cd "d:\【代码】\【Python】\机器学习\歧义侦探"
streamlit run app.py
```

### 步骤3：访问应用
应用启动后，在浏览器中打开：
- **本地地址**: http://localhost:8502
- **网络地址**: http://139.226.128.44:8502

## 🔧 常见问题解决

### 问题1：提示"ModuleNotFoundError: No module named 'spacy'"
**解决方案**：
```bash
pip install spacy
python -m spacy download en_core_web_sm
```

### 问题2：提示"Resource 'punkt' not found"
**解决方案**：
```bash
python -c "import nltk; nltk.download('punkt')"
```

### 问题3：提示"Resource 'averaged_perceptron_tagger' not found"
**解决方案**：
```bash
python -c "import nltk; nltk.download('averaged_perceptron_tagger')"
```

### 问题4：端口8502被占用
**解决方案**：
应用会自动使用其他端口（如8503、8504等），查看终端输出的实际端口号。

### 问题5：浏览器无法打开
**解决方案**：
1. 检查防火墙设置
2. 尝试使用 http://127.0.0.1:8502
3. 确认Streamlit进程正在运行

## 📝 使用说明

### 1. 输入句子
在页面顶部的输入框中输入要分析的英文句子，默认句子是：
"The boy saw the man with the telescope."

### 2. 查看分析结果
- **依存关系标签页**：查看依存句法树的可视化图表
- **成分结构标签页**：查看成分句法树和词性标注

### 3. 核心论元提取
页面底部会自动显示提取的核心论元：
- Root（根节点）
- 主语 (nsubj)
- 直接宾语 (dobj)
- 介词宾语 (pobj)

### 4. 歧义分析实验
按照页面提示输入经典歧义句子，观察模型的分析结果。
